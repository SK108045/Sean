## Overview

Developed a sophisticated load balancing suite demonstrating four distinct distribution algorithms using NGINX and Node.js. The system includes comprehensive testing frameworks and real-time performance monitoring for each algorithm implementation.
- **📦 [View Project on GitHub](https://github.com/SK108045/Load-Balancing-with-NGINX)**

## Key Features

- **Round Robin Distribution**: Implemented circular request distribution across server pool
- **Least Connections**: Dynamic traffic routing based on server load and response times
- **IP Hash**: Session persistence through client IP-based routing
- **Weighted Round Robin**: Proportional load distribution with configurable server weights
- **Automated Testing**: PowerShell scripts for performance validation
- **Real-time Monitoring**: Request counting and distribution analysis

## Technologies Used

- **NGINX**: Core load balancing implementation
- **Node.js**: Backend server infrastructure
- **PowerShell**: Automated testing scripts
- **HTTP/HTTPS**: Protocol handling
- **Bash**: Server management scripts
- **Git**: Version control
- **Linux**: Deployment environment

## Technical Implementation

- **Algorithm Design**: Built four distinct load balancing strategies
- **Testing Framework**: Created comprehensive testing suite for each algorithm
- **Performance Metrics**: Implemented request counting and distribution analysis
- **Session Management**: IP-based persistence configuration
- **Server Pool**: Multiple Node.js instances with different configurations
- **Load Simulation**: Automated request generation for testing

## Outcome

Successfully demonstrated all four load balancing algorithms with measurable performance metrics. The system effectively distributes traffic according to each algorithm's specific rules, maintaining high availability and optimal resource utilization.
