## Overview

Developed an advanced Twitter bot that leverages Google Vertex AI to generate witty, engaging tweets automatically. The system integrates image search capabilities and automated posting schedules, creating a fully autonomous social media presence.
- **📦 [View Project on GitHub](https://github.com/SK108045/TwitterBot)**

## Key Features

- **AI-Generated Content**: Sophisticated tweet generation using Google Vertex AI
- **Image Integration**: Automatic relevant image search and posting
- **Scheduled Posting**: Automated content distribution using cron jobs
- **Customizable Prompts**: Dynamic content generation system

## Technologies Used

- **Python**: Core programming language
- **Google Vertex AI**: Content generation
- **Tweepy**: Twitter API integration
- **Google Custom Search**: Image sourcing
- **Cron**: Scheduling system
- **Requests**: API handling
- **JSON**: Data formatting

## Technical Implementation

- **AI Integration**: Connected with Google Vertex AI for content creation
- **Image Processing**: Implemented image search and validation
- **Automation**: Set up cron jobs for scheduled posting
- **Error Handling**: Robust error management system
- **API Management**: Efficient handling of rate limits

## Outcome

The bot successfully gained over 300 followers within its first month, demonstrating the effectiveness of AI-generated content. The system maintains consistent engagement while requiring minimal human intervention.
