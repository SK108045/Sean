## Overview

Developed SketchSync, a real-time collaborative drawing platform that enables multiple users to draw, sketch, and create together simultaneously. The platform features instant synchronization across all connected users with a rich set of drawing tools.
- **📦 [View Project on GitHub](https://github.com/SK108045/SketchSync)**

## Key Features

- **Real-time Collaboration**: Instant synchronization of drawings across all users
- **Drawing Tools**: Color picker, adjustable brush sizes, and eraser functionality
- **Firebase Integration**: Real-time database for persistent storage of drawing data
- **WebSocket Communication**: Low-latency data transmission for smooth drawing experience
- **Responsive Design**: Fully adaptive interface across all device sizes

## Technologies Used

- **Node.js**: Server-side runtime environment
- **Express**: Web application framework
- **WebSocket**: Real-time bidirectional communication
- **Firebase**: Real-time database implementation
- **HTML Canvas**: Drawing interface
- **CSS3**: Responsive styling
- **Ngrok**: Tunnel for external access

## Technical Implementation

- **Real-time Sync**: WebSocket implementation for instant drawing updates
- **Data Structure**: Optimized Firebase schema for drawing actions
- **State Management**: Synchronized drawing state across clients
- **Error Handling**: Robust connection and data management
- **Performance**: Optimized canvas rendering and data transmission

## Outcome

Created a highly responsive collaborative drawing platform that enables seamless real-time interaction between multiple users, perfect for remote team brainstorming and creative collaboration.
