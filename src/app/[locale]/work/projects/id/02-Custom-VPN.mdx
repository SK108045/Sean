## Overview

Built a robust VPN solution using OpenVPN Access Server on AWS EC2, providing secure remote access for enterprise users across multiple platforms including Windows, macOS, Linux, Android, and iOS.
- **📦 [View Project on GitHub](https://github.com/SK108045/Custom-VPN)**

## Key Features

- **AWS Infrastructure**: Deployed on EC2 with automated setup scripts and security groups
- **Multi-Platform Support**: Comprehensive support for all major operating systems
- **Security Features**: Advanced IP filtering, military-grade encryption, and granular access controls
- **Automated Deployment**: Shell scripts for rapid deployment and configuration

## Technologies Used

- **OpenVPN Access Server**: Core VPN technology
- **AWS EC2**: Cloud infrastructure platform
- **Ubuntu Server**: Operating system
- **Shell Scripting**: For automation
- **SSL/TLS**: For encryption
- **UFW Firewall**: Network security
- **systemd**: Service management

## Technical Implementation

- **Server Setup**: Automated EC2 instance deployment with security configurations
- **Network Configuration**: Implemented proper routing and NAT settings
- **Access Control**: User management system with role-based access
- **Monitoring**: System health and connection monitoring
- **Backup**: Automated configuration backup system
